# Node & Angular.JS Starter

## Commands
- npm run lint
- npm run lint:fix


## Features
- ESLint
- ES6 with Babel
- Gulp
- Travis CI

## Node Enviroment Variables
Create a file named ".env" in the root directory and fill its contents as follows.

```ruby
DB_HOST=xxx
DB_NAME=xxx
```
